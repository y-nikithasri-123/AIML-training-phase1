# Titanic Data Analysis Using Python

## Project Overview
This project analyzes Titanic passenger data using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Features
- Data Cleaning
- Missing Value Handling
- Data Visualization
- Survival Analysis
- Gender Analysis
- Passenger Class Analysis

## Dataset
Titanic Dataset CSV

## Conclusion
This project identifies factors affecting passenger survival using data analysis techniques.
